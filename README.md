# JavaScript Roadmap Projects

A collection of JavaScript projects completed as part of the **JavaScript Roadmap** on [Roadmap.sh](https://roadmap.sh/javascript). Each project focuses on strengthening core JavaScript concepts through small, practical exercises.

## About

This repository contains my solutions to the JavaScript projects from Roadmap.sh. The goal is to improve my understanding of modern JavaScript by building progressively more challenging projects while following best practices such as:

* Writing reusable functions
* Keeping code modular and readable
* Separating business logic from presentation
* Working with arrays and objects
* Practicing immutable data manipulation
* Improving problem-solving skills

Each project is organized in its own folder and includes:

* Source code (`main.js`)
* A detailed `README.md`
* Sample inputs and outputs
* Explanation of the implemented functions
